Install
------------------------------------------------------------------------------

``s3dbsaver`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install s3dbsaver

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade s3dbsaver
